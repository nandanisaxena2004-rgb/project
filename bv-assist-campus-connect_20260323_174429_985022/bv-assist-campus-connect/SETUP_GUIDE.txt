BV ASSIST: CAMPUS CONNECT - QUICK START GUIDE

INSTALLATION:
1. Extract to: C:\xampp\htdocs\bv-assist-campus-connect
2. Open http://localhost/phpmyadmin
3. Go to Import tab
4. Select database/schema.sql file
5. Click Go
6. Start Apache and MySQL
7. Open: http://localhost/bv-assist-campus-connect

TEST CREDENTIALS:
- Student: student@college.com / student123
- Faculty: faculty@college.com / faculty123
- Admin: admin@college.com / admin123

FOLDER STRUCTURE:
- index.php = Login page (entry point)
- dashboard.php = Role-based redirect
- student-dashboard.php = Student interface
- faculty-dashboard.php = Faculty interface
- admin-dashboard.php = Admin interface
- config/ = Database configuration
- backend/ = Processing files
- assets/ = CSS and JavaScript
- database/ = MySQL schema
- uploads/ = Uploaded files storage

TROUBLESHOOTING:
- "Cannot connect to database" → Check MySQL running
- "Database not found" → Import database/schema.sql in phpMyAdmin
- "404 Page not found" → Check XAMPP path and folder name
- "Session failed" → Clear browser cookies

FEATURES:
Student: View notifications, download handouts, submit feedback
Faculty: Upload handouts, post notifications, view student feedback
Admin: Manage users, view files, manage system
