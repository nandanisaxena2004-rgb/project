# BV Assist: Campus Connect - College Management System

A complete college management system built with PHP, MySQL, HTML, CSS, and JavaScript.

## 📋 Prerequisites
- PHP 7.4 or higher
- MySQL 5.7 or higher
- XAMPP or similar local server

## 🚀 Installation

1. Extract files to XAMPP htdocs folder
2. Open phpMyAdmin and import database/schema.sql
3. Start Apache and MySQL
4. Access: http://localhost/bv-assist-campus-connect

## 📁 Project Structure
- index.php: Login page
- dashboard.php: Role-based redirect
- student-dashboard.php: Student interface
- faculty-dashboard.php: Faculty interface
- admin-dashboard.php: Admin interface
- config/: Database and session configuration
- backend/: PHP processing files
- assets/: CSS and JavaScript files
- database/: MySQL schema and sample data
- uploads/: Uploaded files storage
